# NextJS Portfolio

A minimal, clean portfolio template built with Next.js.

## Features

- 🚀 Fast and lightweight
- 📱 Fully responsive
- 🔍 SEO friendly
- 🌙 Dark/Light mode
- 🎨 Clean, minimal design
- ⚡ Easy to customize
- 🚦 Next.js App Router ready
- 📊 Analytics-ready (add via `layout.js`)

## Tech Stack

- Next.js 15
- Tailwind CSS
- JavaScript
- DaisyUI
